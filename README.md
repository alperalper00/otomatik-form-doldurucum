🇬🇧 English:

Automated Google Form filler — bulk submit responses to any Google Form with custom or randomized data.

🇹🇷 Türkçe:

Google Form otomatik doldurma botu — herhangi bir Google Form'a toplu ve özelleştirilebilir yanıtlar gönder.


🔧 Installation
Chrome

Go to chrome://extensions/
Enable "Developer mode" from the top right
Click "Load unpacked"
Select the downloaded folder

Edge

Go to edge://extensions/
Enable "Developer mode" from the bottom left
Click "Load unpacked"
Select the downloaded folder

Firefox

Go to about:debugging#/runtime/this-firefox
Click "Load Temporary Add-on"
Select the manifest.json file inside the downloaded folder


⚠️ Note: Firefox temporary installs reset on every browser restart. For permanent installation, the extension must be published on Firefox Add-ons store.

Brave

Go to brave://extensions/
Enable "Developer mode" from the top right
Click "Load unpacked"
Select the downloaded folder


Brave is Chromium-based, so it follows the same steps as Chrome.

🔧 Kurulum
Chrome

chrome://extensions/ adresine git
Sağ üstten "Geliştirici modu"nu aç
"Paketlenmemiş öğe yükle" butonuna tıkla
İndirdiğin klasörü seç

Edge

edge://extensions/ adresine git
Sol alttaki "Geliştirici modu"nu aç
"Paketlenmemiş uzantı yükle" butonuna tıkla
İndirdiğin klasörü seç

Firefox

about:debugging#/runtime/this-firefox adresine git
"Geçici Eklenti Yükle" butonuna tıkla
İndirdiğin klasördeki manifest.json dosyasını seç


⚠️ Not: Firefox'ta geçici yükleme her tarayıcı kapanışında sıfırlanır. Kalıcı kurulum için eklentinin Firefox Add-ons mağazasında yayınlanması gerekir.

Brave

brave://extensions/ adresine git
Sağ üstten "Geliştirici modu"nu aç
"Paketlenmemiş öğe yükle" butonuna tıkla
İndirdiğin klasörü seç


Brave, Chromium tabanlı olduğu için Chrome ile aynı adımları izler.
